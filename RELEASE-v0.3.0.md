# v0.3.0 – Project Variant System

## Added

- Formal Adolar project variant system
- `variants/manifest.json` as the canonical variant registry
- Variant README files for Adolar, Radio, Disco, TV, Player and Core
- Template for future project variants
- CSS design tokens for project variants
- Variant gallery example page
- Social preview SVG templates for all variants
- Documentation for naming, colors, symbols and rules

## Notes

This release defines how future Adolar subprojects should visually connect to the MZ-1 brand system.

The main rule remains:

> Same rocket. Different mission.
