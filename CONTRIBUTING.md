# Contributing

Danke für dein Interesse am Adolar-Branding.

## Grundregeln

- Die MZ-1-Silhouette bleibt gleich.
- Keine Comic-Augen, kein Mund, keine Figur.
- Keine starken 3D-Effekte.
- Keine aggressiven Raketen-/Militär-Assoziationen.
- Orbitron bleibt die Schriftbasis.
- Projektvarianten entstehen über Akzentfarbe, Cockpit und Zustand, nicht über neue Raketenformen.

## Pull Requests

Bitte bei Änderungen an Logo oder Rakete immer eine Vorschau als PNG beilegen.
