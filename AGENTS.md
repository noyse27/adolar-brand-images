# AGENTS.md - Adolar Brand Images

## Projektkontext

Dieses Repository enthält die Brand Assets für das Adolar-Universum.

Die zentrale visuelle Identität ist die Rakete **MZ-1**.  
MZ-1 ist das gemeinsame Logo-/Maskottchen-System für Adolar, Adolar Radio, Adolar Disco, Adolar TV und weitere Projekte.

## Wichtige Designregeln

Bitte unbedingt einhalten:

- Keine SVGs durch komplett andere Designs ersetzen.
- Keine zufälligen AI-/Icon-Assets hinzufügen.
- Keine Font-Dateien ungeprüft ins Repo packen.
- Die Orbitron-Abhängigkeit nicht entfernen.
- Die MZ-1-Silhouette nicht ändern.
- Keine Comic-Augen, kein Mund, kein Gesicht.
- Keine militärische oder aggressive Raketenoptik.
- SVGs bleiben die Source of Truth.
- PNGs, Favicons, ICOs und Social Previews sind generierte Exporte.

## Was geändert werden darf

Codex darf:

- Build-Skripte reparieren oder verbessern.
- GitHub Actions reparieren oder verbessern.
- Export-Automation verbessern.
- README, Docs und Beispiele aktualisieren.
- CSS/JS für Animationen verbessern.
- Fehlende Exporte generieren.
- Dateistruktur konsistenter machen.

## Was nicht geändert werden soll

Codex soll nicht:

- eine neue Logo-Richtung entwerfen.
- die MZ-1-Rakete neu zeichnen.
- Orbitron durch eine andere Schrift ersetzen.
- generische Icon-Sets einbauen.
- externe Font-Dateien ohne Lizenzprüfung hinzufügen.
- bestehende Brand-Regeln abschwächen.
