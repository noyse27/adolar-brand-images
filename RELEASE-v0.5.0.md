# v0.5.0 – Build / Export Automation

## Added

- Node-based build script for generated brand exports
- Repository sanity check script
- `package.json` with build/check commands
- GitHub Actions workflow for building exports
- Favicon manifest generation
- PNG export generation from SVG sources
- Social preview PNG generation
- Build automation documentation

## Notes

SVG files remain the source of truth. Generated PNGs, favicons and social previews can now be rebuilt consistently.

ICO generation is prepared via source PNGs and can be expanded in a later release.
