# Adolar Brand Variant

Master brand for the Adolar universe.

## Brand Tokens

| Token | Wert |
|---|---|
| Slug | `adolar` |
| Akzentfarbe | `#31D5FF` |
| Cockpit-Symbol | `keines` |

## Verwendung

Diese Variante darf für Webseiten, App-Icons, Splashscreens und Projekt-READMEs verwendet werden.

Die MZ-1-Rakete bleibt dieselbe wie in allen anderen Adolar-Projekten.
