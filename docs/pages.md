# GitHub Pages Preview

The static preview lives in `site/`:

```text
site/
  index.html
  styles.css
  app.js
```

It presents the existing MZ-1 assets only: master logo, rocket, variant icons, animation states, palette, and generated export paths. It does not introduce a new design direction.

To preview locally, open `site/index.html` in a browser from the repository checkout. If GitHub Pages is enabled later, configure the Pages source to publish the `site/` directory.
