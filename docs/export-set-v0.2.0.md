# Export Set v0.2.0

Dieses Paket ergänzt das Brand-Repository um praktische Export-Dateien.

## Enthalten

- Varianten für:
  - Adolar
  - Adolar Radio
  - Adolar Disco
  - Adolar TV
- App-Icon-SVGs
- transparente Icon-SVGs
- PNG-Exporte in Standardgrößen
- Windows `.ico`
- Favicon-Dateien
- GitHub Social Preview

## Hinweis zu Orbitron

SVG-Dateien verwenden weiterhin echten Text mit `font-family: "Orbitron"`.
Für finale Logo-PNGs mit exakt identischer Typografie sollte später eine Pfad-Version des Wortmarks erzeugt werden.

## Variantenregel

Die MZ-1-Rakete bleibt die verbindende Form.
Projektvarianten unterscheiden sich nur durch:

- Akzentfarbe
- Cockpit-Symbol
- Produktname im Logo
