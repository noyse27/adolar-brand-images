# Adolar Design Language – MZ-1

## Grundprinzip

Die Rakete ist das gemeinsame Erkennungszeichen des Adolar-Universums.

Adolar, Adolar Radio, Adolar Disco und spätere Projekte sollen dieselbe Rakete verwenden. Unterschiede entstehen über Akzentfarben, Cockpit-Zustände oder kleine Projektvarianten – nicht über eine neue Raketenform.

## Charakter

MZ-1 soll freundlich, retro-futuristisch und zeitlos wirken.

Nicht:
- aggressiv
- militärisch
- cyberpunk
- überladen
- fotorealistisch

Ja:
- rund
- sympathisch
- klar
- vektorfähig
- bei kleinen Größen erkennbar

## Typografie

Orbitron bleibt die Projektschrift. Der Name ADOLAR wird ohne Untertitel gesetzt.

## Zustände

| Zustand | Verhalten |
|---|---|
| `idle` | Rakete schwebt ruhig |
| `loading` | Flamme pulsiert langsam |
| `playing` | Flamme pulsiert schneller |
| `paused` | Flamme fast aus |
| `error` | Rakete kippt leicht, Cockpit-Akzent rot |

## Regel

Die MZ-1-Silhouette bleibt gleich. Neue Projekte bekommen keine neue Rakete, sondern eine neue Anwendung derselben Rakete.
